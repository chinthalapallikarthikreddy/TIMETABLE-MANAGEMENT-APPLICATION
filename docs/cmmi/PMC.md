# Project Monitoring and Control (PMC)

## Purpose
Project Monitoring and Control ensures progress is tracked against plan and corrective actions are taken when deviations occur.

## Application in Project
- Sprint burndown charts monitored progress.
- EVM metrics tracked cost and schedule performance.
- Sprint reviews identified corrective actions.
