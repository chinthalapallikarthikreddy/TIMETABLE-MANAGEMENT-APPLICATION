# Project Planning (PP)

## Purpose
Project Planning establishes estimates, schedules, resources, and commitments for successful project execution.

## Application in Project
- Work was divided into 4 sprints.
- Story points were estimated using Planning Poker.
- Sprint milestones and timelines were defined in GitHub.

## Evidence
- Sprint Planning Board
- PERT Activity List
- Story Point Estimates
- Sprint Milestones
